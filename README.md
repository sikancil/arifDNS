### arifDNS
###### node.js DNS Server build using native-dns
###### https://plus.google.com/+ArifWidianto
------
Current version 0.1.0-alpha
------
Dependencies:
* native-dns: 0.6.1 (install using node v0.10.32)
* winston: ^0.8.1
* nodejs: 0.11.14 (running)

Service types:
* SOA
* A
* AAAA
* NS
* CNAME
* MX
* TXT
* SRV
* PTR (later..)
* NAPTR (later..)
